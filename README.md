### Newton Physics

Experimental Newtonian dynamics engine written in Rust and visualized with kiss3d.

---
### Installation
```bash
git clone https://github.com/quthery/newton-physics
cd newton-physics
cargo run --release
```
---
### Goals
- Realize newtonian mechanics
- Rewrite to 3D
- Rewrite calculating to *Verlet Integration*
---
### Tech Stack
- Rust
- kiss3d
- nalgebra
